📡 시스템 구조

Mac 또는 Windows PC에서 서버를 실행하면, 같은 Wi-Fi에 연결된 모든 기기(iPad, Android, 다른 PC)에서 브라우저로 접속할 수 있습니다. 별도 앱 설치가 필요 없습니다!

🔗 접속 주소

서버 PC의 IP 주소를 아래 형식으로 브라우저에 입력하세요

POS (계산대) http://서버IP:8080/pos
Admin (관리자) http://서버IP:8080/admin
Order (고객 주문 키오스크) http://서버IP:8080/order
Kitchen (주방 디스플레이) http://서버IP:8080/kitchen
💡 서버 IP 확인 방법: 서버를 시작하면 터미널에 IP 주소가 표시됩니다.
예: http://192.168.1.100:8080

📱 기기별 접속 방법

🍎

iPad / iPhone

Safari 브라우저에서 접속 주소를 입력하세요

🤖

Android

Chrome 브라우저에서 접속 주소를 입력하세요

🍎 iPad 홈 화면에 바로가기 추가

1
Safari에서 접속 주소 열기
2
하단 공유 버튼 (□↑) 탭
3
"홈 화면에 추가" 선택
4
이름 입력 (예: "The Bap POS") → 추가
✨ 전체 화면 모드: 홈 화면 바로가기로 열면 주소창 없이 앱처럼 전체 화면으로 실행됩니다!

🤖 Android 홈 화면에 바로가기 추가

1
Chrome에서 접속 주소 열기
2
우측 상단 ⋮ 메뉴 탭
3
"홈 화면에 추가" 선택
4
이름 입력 → 추가

🏪 지점별 추천 기기 배치

위치 화면 추천 기기
계산대 POS iPad 10.2" 이상 / Windows PC
카운터 앞 고객 주문 iPad / Android 태블릿
주방 주방 디스플레이 태블릿 / 모니터 + PC
사무실 관리자 Mac / Windows PC

🔧 문제 해결

접속이 안 될 때

1
서버 PC와 같은 Wi-Fi에 연결되어 있는지 확인
2
서버가 실행 중인지 확인 (터미널에 "Server running" 표시)
3
방화벽에서 포트 8080이 열려있는지 확인
4
IP 주소가 맞는지 확인 (서버 재시작 시 IP가 변경될 수 있음)

Mac 방화벽 설정

시스템 설정 → 네트워크 → 방화벽 → Node.js 허용
또는 터미널: sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/local/bin/node

Windows 방화벽 설정

Windows 보안 → 방화벽 → 고급 설정 → 인바운드 규칙 → 새 규칙 → 포트 8080 허용
