📝 통신 로그
📖 사용 방법
1.

서버 실행: node tb-server.js

2.

이 페이지를 http://[서버IP]:8080/test 로 열기 (file:// 금지!)

3.

"서버 상태 확인" 클릭 → HTTP 연결 확인

4.

"주문쪽 연결" → "주방쪽 연결" 순서대로 클릭

5.

"WebSocket으로 주문 전송" 클릭 → 주방쪽에 도착하는지 확인

⚠️

다른 기기 테스트: 같은 WiFi에서 서버 맥의 IP로 접속 (예: http://192.168.1.5:8080/test)
