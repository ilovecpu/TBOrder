================================================================================
                         ORDER MANAGEMENT
================================================================================

ORDER NUMBER GENERATION
  Format: TB-NNN (e.g., TB-001, TB-002)
  Daily Reset: Counter resets at midnight (new day)
  Uniqueness: Per-day sequential numbering

ORDER STORAGE
  Database: IndexedDB (TheBapKiosk)
  Store: orders
  Retention: Indefinite (local)
  Sync: Ready for backend integration

CART PERSISTENCE
  Database: IndexedDB (TheBapKiosk)
  Store: cart
  Auto-load: On page initialization
  Auto-save: After every change
  Clear: On order confirmation

                       QUICK REFERENCE

TO OPEN:
  1. Download TBOrder_Kiosk.html
  2. Open in web browser
  3. Select service type
  4. Begin ordering

TO CUSTOMIZE COLORS:
  - Find CSS variables at top of <style> section
  - Change: --pink-primary, --green, --dark, etc.

  - Edit MENU_DATA object in JavaScript
  - Add entry with all required fields

  - Search for "The Bap" in HTML
  - Replace with your restaurant name
