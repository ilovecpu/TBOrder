📋 설정 가이드 (Setup Guide)

  1. Google Sheets 생성 — sheets.google.com에서 새 스프레드시트 만들기
  2. 시트 이름 변경 — 기본 시트 이름을 Orders로 변경
  3. Apps Script 열기 — 메뉴: 확장프로그램 → Apps Script 클릭
  4. 코드 붙여넣기 — google-apps-script.js 내용을 전부 복사해서 붙여넣기
  5. 배포 — 배포 → 새 배포 → 웹 앱 선택
    • "다음 계정으로 실행" → 나
    • "액세스 권한" → 모든 사용자
  6. URL 복사 — 배포 후 나오는 URL을 복사
  7. ⚠️ URL 적용 (중요!)
    • mobile.html 파일의 APPS_SCRIPT_URL에 붙여넣기
    • TBKitchen_Kiosk.html 파일의 APPS_SCRIPT_URL에 붙여넣기
  8. GitHub에 push — 변경된 파일을 GitHub에 push하면 완료!

🔗 GitHub Pages URL 설정

QR코드가 연결될 기본 URL을 입력하세요. GitHub Pages URL 형식: https://사용자명.github.io/TBOrder/mobile.html
